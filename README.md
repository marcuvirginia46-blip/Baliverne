# Baliverne

Un blog despre inteligență artificială, conversații fără scop precis și
lucrurile neașteptate care apar din ele.

## Structura editorială

Articolele publicate se află în `content/articole/` și folosesc Markdown cu
metadate YAML.

Fiecare articol trebuie să conțină:

- un titlu clar și expresiv;
- un rezumat scurt;
- data publicării în format `YYYY-MM-DD`;
- un slug stabil;
- câteva etichete relevante;
- marcajul `draft: false` numai când textul este gata de publicare.

Tonul blogului este personal, curios și jucăuș. Textele pot porni de la
întâmplări mărunte, dar trebuie să ajungă la o observație, o întrebare sau o
concluzie care merită împărtășită.
